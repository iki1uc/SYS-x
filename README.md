# SYS-x – System-Kreuzung (iki1uc)

SYS-x ist die Cross-Ebene des iki1uc-Systems.
Es verbindet:

- PX3 / PX6 / PX9 (PX-TRIO)
- PX12 / PX4 (Fusion + Achse)
- SYS / TEM (System + Zeit)
- RESPO (Response-Achse)
- GEO (O12 / Φ4 / XU / TΩ)
- UI (UI-SCANNER / RUN8)

SYS-x erzeugt:
- System-Relevanz
- Achsen-Synchronisation
- Orbit-Neutralisierung
- Root-Bindung
